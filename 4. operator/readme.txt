
// In JavaScript, there are several types of operators that allow you to perform different operations on values. These operators can be categorized into the following main types:

// 1. **Arithmetic Operators**: These operators are used for mathematical calculations.
//    - Addition +
//    - Subtraction -
//    - Multiplication *
//    - Division /
//    - Modulus % (returns the remainder of a division)
//    - Increment ++ (increases a variable by 1)
//    - Decrement -- (decreases a variable by 1)

// 2. **Assignment Operators**: These operators are used to assign values to variables.
//    - Assignment =
//    - Addition Assignment +=
//    - Subtraction Assignment -=
//    - Multiplication Assignment *=
//    - Division Assignment /=
//    - Modulus Assignment %=

// 3. **Comparison Operators**: These operators are used to compare values and return a Boolean result (true or false).
//    - Equal to ==
//    - Not equal to !=
//    - Strict equal to ===
//    - Strict not equal to !==
//    - Greater than >
//    - Less than <
//    - Greater than or equal to >=
//    - Less than or equal to <=

// 4. **Logical Operators**: These operators are used to combine or manipulate Boolean values.
//    - Logical AND &&
//    - Logical OR ||
//    - Logical NOT !

// 5. **Conditional (Ternary) Operator**: This operator provides a shorthand way of writing conditional statements.
//    - Conditional ? :

// 6. **Bitwise Operators**: These operators perform bit-level operations on integers.
//    - Bitwise AND &
//    - Bitwise OR |
//    - Bitwise XOR ^
//    - Bitwise NOT ~
//    - Left Shift <<
//    - Right Shift >>
//    - Zero-fill Right Shift >>>

// 7. **Unary Operators**: These operators operate on a single operand.
//    - Unary plus +
//    - Unary minus -
//    - typeof (returns the data type of an operand)
//    - void (discards a return value)
//    - delete (deletes an object's property)

// 8. **Other Operators**:
//    - Comma , (used to separate expressions)
//    - instanceof (checks if an object is an instance of a specific class)
//    - in (checks if an object has a certain property)
